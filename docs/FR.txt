Функциональные требования к Интернет Банкинг Системе:

Пользователь:

	Создать учётную запись
		Заполнить форму регистрации, ввести e-mail, password, заполнить captcha.
		
	Связать аккаунт с карточкой
		Ввести номер карточки, подтвердить полученным по почте ключом принадлежность карточки.
		При создании карточки, администратор укажет e-mail, на который зарегистрирована карточка.

	Работа с секцией "Платежи"
		*Оплатить на конкретный карт-счёт
		ОФормить новый платёж, выбрав получателя платежа, указав сумму и данные платежа. По окончанию операции можно добавить платёж в "Сохранённые".		
		Оформить платёж, выбрав его из "Сохранённых платежей".
		Возможность удалить платёж из списка сохранённых платежей.
		Возможность просматривать историю совершённых платежей.	
		Возможность из списка совершённых платежей в "Истории платежей" перенести информацию в "Сохраненные платежи".
		
	Работа с секцией "Карты"
		Выводится список карт пользователя. 
		По каждой карте следующая информация: 
			остаток средств, валюта счёта, номер, примечания, *тип.
		Нажав на кнопку "Выписка по карте" можно увидеть список операций за выбранный период.
		Можно настроить автооплату на указанный счёт.
		Оставить заявку на карту:
			Заполнить ФИО, паспортные данные и выбрать карточку. 
			Заявка поступает администратору, после чего он создаёт карточку для заявителя.
		
	Работа с секцией "Переводы"
		Перевести денежные средства с одного карт счета на другой
		История переводов
		
		Примечание.
		В случае если валюта транзакции отличается от валюты счета, 
		операция перевода средств с карточки на карточку выполняется ТОЛЬКО через белорусские рубли. 
		При выполнении перевода средств по карточкам их списание и зачисление производятся по курсам покупки и продажи соответственно,
		установленным на день выполнения операции.

		В случае если иностранная валюта транзакции совпадает с валютой счета, 
		операция перевода средств с карточки на карточку выполняется напрямую в данной иностранной валюте без конверсии. 	

		Минимальные суммы перевода в зависимости от выбранной валюты составляют
		5000 BYR, 1 USD, 1 EUR и 30 RUB соответственно.
		
Администратор:
	
	Добавить/удалить/редактировать банковскую карточку(и) в системе
	Вводить курсы валют, если недоступен сервис национального банка
	Пополнение карт счета клиента
	Отмена операций
	
	
* - помечено то, насчёт чего пока не определились.
